# Автоматизація Відео-Воркфлоу (After Effects)

Цей документ описує два професійні воркфлоу для автоматизації створення контенту (Hook Swapper) та локалізації (Localization Render Pipeline).

---

## Задача 1: Hook Swapper (Тестування Хуків)

**Мета**: Автоматичне створення варіацій відео шляхом поєднання різних "Хуків" (3–5 сек) з одним "Майстер-відео" (Body + CTA).

### Воркфлоу:
1. **Підготовка**: Помістіть 5 хук-кліпів в одну папку (наприклад, `hooks/`).
2. **Запуск**: Відкрийте [hook_swapper.jsx](file:///e:/Teach/Scelar/Scelar2026/MyScripts/hook_swapper.jsx) в After Effects.
3. **Налаштування**:
   - Виберіть файл майстер-відео.
   - Виберіть папку з хуками.
4. **Результат**:
   - Скрипт створить нову композицію для кожної варіації.
   - Хук ставиться на початок (0:00), а майстер-відео автоматично зміщується на час завершення хука.
   - Усі варіації додаються в Render Queue для миттєвого рендеру.

**Очікуваний результат**: 5 готових до рендеру відеофайлів, кожен з унікальним початком, але однаковим основним повідомленням та CTA.

---

## Задача 2: Localization Render Pipeline (Локалізація)

**Мета**: Автоматична генерація мовних версій відео на основі таблиці даних (CSV).

### Воркфлоу:
1. **Підготовка**: Створіть CSV-файл з колонками: `lang`, `headline`, `subtitle`, `cta`, `voiceover_path`.
2. **Проєкт AE**: Переконайтеся, що назви текстових шарів у вашій композиції відповідають назвам колонок у CSV (наприклад, шар називається "headline").
3. **Запуск**: Відкрийте [localization_pipeline.jsx](file:///e:/Teach/Scelar/Scelar2026/MyScripts/localization_pipeline.jsx).
4. **Налаштування**:
   - Виберіть CSV-файл.
   - Вкажіть назву майстер-композиції.
5. **Результат**:
   - Скрипт дублює майстер-композицію для кожної мови.
   - Автоматично замінює текст у шарах на відповідний переклад з CSV.
   - Імпортує та додає файл дикторської озвучки (Voiceover), якщо шлях вказано в таблиці.
   - Додає всі мовні версії в Render Queue.

**Очікуваний результат**: Повністю локалізовані відео для кожної мови з таблиці, готові до експорту без ручної роботи.

## Використання Штучного Інтелекту (AI)
Для покращення цих воркфлоу рекомендується:
- **Транскрибація та переклад**: Використовуйте [transcribe.py](file:///e:/Teach/Scelar/Scelar2026/MyScripts/transcribe.py) (Whisper) для отримання тексту з хуків та GPT-4 для автоматичного заповнення CSV-таблиці локалізації.
- **Озвучка (TTS)**: Використовуйте API ElevenLabs або OpenAI TTS для автоматичної генерації `voiceover_path` файлів перед запуском рендеру.
