# Набор инструментов для продвинутой автоматизации видео (Task 2)

Коллекция профессиональных инструментов автоматизации для After Effects и создания контента на базе ИИ, разработанная для высокоуровневых рабочих процессов.

## 🚀 Основные возможности

### 1. Hook Swapper (Тестирование хуков)
- **hook_swapper.jsx**: Автоматически создает несколько вариаций видео, объединяя различные "Хуки" (3-5 сек) с основным видео (Master Body + CTA).
- **hook_generator.py**: Использует OpenAI/GPT-4o для анализа транскрибации и генерации 5 виральных заголовков и визуальных описаний для хуков.

### 2. Localization Pipeline (Локализация)
- **localization_pipeline.jsx**: Генерирует локализованные версии видео на основе конфигурации из CSV-файла.
- **tts_generator.py**: Интегрируется с ElevenLabs API для генерации эмоциональной многоязычной озвучки для процесса локализации.

### 3. Улучшение качества с помощью ИИ
- **video_upscale.py**: Использует **Real-ESRGAN** для высококачественного покадрового апскейлинга UGC-клипов до 4K.
- **comfy_bridge.py**: Соединяет After Effects с локальным или удаленным экземпляром **ComfyUI** для генеративных задач (дорисовка фона, стилизация).

## 🛠 Установка и настройка

### Предварительные требования
1. **Adobe After Effects** (любая современная версия)
2. **Python 3.10+**
3. **ComfyUI** (опционально, для продвинутых задач генерации)

### Зависимости Python
```bash
pip install requests openai websocket-client pandas torch basicsr realesrgan opencv-python
```

### Установка
1. Скопируйте файлы `.jsx` в папку After Effects `Scripts/ScriptUI Panels`.
2. Перезапустите After Effects.
3. Доступ к инструментам осуществляется через меню `Window`.

## 👨‍💻 Рекомендации Senior-разработчика
- **Динамические ассеты**: Всегда используйте абсолютные пути в CSV-конфигурациях, чтобы избежать ошибок импорта.
- **Безопасность ИИ**: Храните API-ключи в переменных окружения или в защищенном файле `.env`, не прописывайте их в коде.
- **Масштабируемость**: Для обработки больших пакетов данных используйте скрипт `video_upscale.py` на выделенном GPU-сервере.
